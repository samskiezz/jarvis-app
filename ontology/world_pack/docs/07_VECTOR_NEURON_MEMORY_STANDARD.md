# Vector Neuron Memory Standard

## Definition
A neuron/memory node is an engineering abstraction. It is not biological. It is a controlled memory unit consisting of:
- vector embedding
- graph node
- metadata record
- source provenance
- ontology target
- policy constraints
- audit evidence
- feature functions

## Required fields
- neuron_id
- neuron_type
- vector_namespace
- embedding_record_id
- embedding_text_template
- source_reference_ids
- ontology_targets
- feature_functions
- policy_controls
- audit_records

## Retrieval rule
AI context retrieval must filter by:
- tenant
- purpose
- classification
- object permission
- property permission
- source licence
- export control
- redaction policy
