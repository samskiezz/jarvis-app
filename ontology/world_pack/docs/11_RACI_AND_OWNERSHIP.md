# RACI and Ownership

## Roles
- Platform Owner
- Data Product Owner
- Source Steward
- Ontology Steward
- Security Officer
- AI Governance Lead
- Pipeline Engineer
- Graph Engineer
- Vector/Retrieval Engineer
- Mission App Owner
- Auditor

## RACI summary
- Source registration: Source Steward accountable.
- Licence review: Security/Legal accountable.
- Parser implementation: Pipeline Engineer responsible.
- Ontology mapping: Ontology Steward accountable.
- Graph edges: Graph Engineer responsible.
- Vector memory: Retrieval Engineer responsible.
- AI context: AI Governance Lead accountable.
- Production release: Platform Owner accountable.
- Audit evidence: Auditor accountable.
