# AIP AI Governance Standard

## AI boundary
AI does not access raw data directly. It receives governed context from the ontology, graph, vector and evidence services.

## Required AI controls
- prompt registry
- model registry
- tool registry
- agent registry
- context builder
- permission filtering
- citation enforcement
- redaction
- trace logging
- evals
- human approval for sensitive actions

## Evaluation metrics
- groundedness
- citation accuracy
- retrieval precision/recall
- policy compliance
- sensitive-data leakage
- tool-call correctness
- action safety
- latency
- cost
- outcome quality
