# What Needs Image vs GLB/PBR vs Animation

## Images
Use for backgrounds, panel skins, textures, icons, sprites, visual references, and low-cost static art.

## GLB/PBR
Use for floating 3D props: neural brain, map, GPU/server hardware, pipeline nodes, inference ladder, knowledge graph objects, mic orb, Claude workspace props, Asset Forge machine.

## Animation/VFX
Use for interaction, data updates, voice, alerts, routing particles, progress, state transitions, and visual feedback.

## Live components
All numbers, charts, table rows, buttons, toggles, forms, text labels, and percentages must be rendered as live UI components.
