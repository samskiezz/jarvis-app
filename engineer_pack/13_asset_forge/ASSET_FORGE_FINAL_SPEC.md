# Asset Forge Final Spec

## Widget modes

- Text to Image.
- Image Edit.
- Text to GLB.
- Image to GLB.
- Multi-view to GLB, optional.
- Optimize GLB.
- Add to Scene.

## UI controls

Generate, Regenerate, Edit Prompt, Save Asset, Add to Scene, Replace Existing, Download, Delete/Archive, Mark Approved.

## Asset states

draft -> generated -> reviewed -> approved -> live -> archived

## Required metadata

asset_id, name, type, prompt, provider, model, output URL, thumbnail URL, GLB URL, texture URLs, PBR enabled, created_at, created_by, target panel, scene placement preset, approval state, cost estimate.
