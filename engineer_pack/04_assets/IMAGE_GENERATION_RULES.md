# GPT Image 2 Generation Rules

1. Generate image assets at the highest available quality suitable for UI textures/reference.
2. Do not bake dynamic data or UI text into images.
3. Use transparent PNG/WebP when asset is an icon/sprite/overlay.
4. Use 16:9 or ultrawide for backgrounds; square for icons; tileable for textures.
5. Save prompt, seed/metadata if available, asset ID, provider, date, and target panel.
6. Every generated image must pass review before becoming live.
