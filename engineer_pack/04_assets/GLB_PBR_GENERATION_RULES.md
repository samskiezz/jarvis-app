# Tripo3D GLB/PBR Generation Rules

1. Generate GLB assets with PBR textures where suitable.
2. Every GLB needs thumbnail, metadata, target panel, scale preset, origin/pivot check, and LOD/compression step.
3. Browser performance matters: optimize geometry, textures, and emissive maps.
4. Use emissive/holographic materials for brain/map/routing props.
5. Use metallic/roughness PBR for GPU/server hardware.
6. Every GLB must pass preview, scale, material, and performance QA before going live.
