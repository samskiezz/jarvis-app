# Jarvis Command Center Demonstrator

This folder is a demonstrator shell only. It is not a claim about how Palantir implements UI.

Production UI requirement:
- framework-agnostic
- driven by typed graph contracts
- every edge backed by evidence/audit
- Foundry/Gotham/Apollo three-plane operator model
- no fake random visual lines
- no ungoverned actions
- Jarvis can only propose governed actions through the action engine

Run locally if using the demonstrator:
```bash
npm install
npm run dev
```
