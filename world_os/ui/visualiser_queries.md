# Visualiser Query Library

## Foundry
- show source lineage for subject_id
- show ontology object fan-out
- show vector memory namespace and retrieved evidence
- show pipeline failures by quality gate

## Gotham
- show live event impact graph
- show entity investigation graph
- show timeline reconstruction
- show evidence chain

## Apollo
- show desired vs current state
- show rollout wave dependencies
- show drift graph
- show rollback blast radius

Every graph edge must be backed by relationship_type, evidence_id, audit_id and confidence.
